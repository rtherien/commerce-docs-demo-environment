## [1.0.0] - 2025-11-25

### Added
- Initial release of Coveo Commerce API Loader
- Support for both Update and Load operations via Coveo Stream API
- Interactive CLI mode for guided operation
- Command-line interface for automation
- Comprehensive payload analysis and validation
- Professional project structure with proper Python packaging
- Virtual environment setup automation
- Configuration management with templates
- Extensive documentation and troubleshooting guides
- Error handling with detailed user feedback
- Support for all major Coveo Commerce object types (Product, Variant, Availability)
- Rate limiting and API error handling
- Payload file management and organization
- Safety confirmations for destructive operations
- Examples and demo data included
